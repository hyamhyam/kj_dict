KJ_browser (プロトタイプ版)

top > KJ_dict マニュアル > KJ_browser

KJ_dictのライブラリ(KJ_dict.dll)には韓日翻訳を行うメソッドがあります。
それを使えば，こんな事もできるという例として作ってみました。

ただし，現在では無料で Webページの韓国語→日本語翻訳を行えるサイトも多々あるので，
KJ_browserの存在意義が薄くなりました。
自分の趣味と勉強のため，細々とメンテするかもしれません。

説明

KJ_browserは韓国語→日本語の翻訳に特化したインターネットブラウザです。

Yahoo Korea IT Newsページに対して使ってみた例。


使い方

KJ_browser.exe を実行すると大昔のブラウザのようなウィンドがあがります。
翻訳したい韓国のサイトへ「go」で移動し、[翻訳] ボタンを押してください。

結構、時間かかります。
プログラムを終わるときは、[終了] を押してください。

注意事項

  • ScriptやActiveXなどがあるサイトではうまく動作しない事があります。
    もし，よく翻訳するサイトがあれば， IEのブラウザのインターネットオプションのセキュリティ設定で
    アクティブスクリプトやActiveXを無効にしておくのがお薦めです。
  • 本格的な翻訳でなく，単なる最長一致単語置換です。
    あくまで実験的なツールです。
    翻訳結果を信用して間違った情報を身につけた、
    なんて苦情は一切受け付けません。 :-P
