脆弱性に関する情報

2005.07.20
「Internet Explorer コンポーネントを使用するアプリケーションにおけるセキュリティゾーンの扱いに関する脆弱性」
http://jvn.jp/jp/JVN%23257C6F28/index.html
というものがあります。

IEコンポーネントを使用し、かつ
(1)セキュリティゾーンをプログラム側で変更している場合。
または
(2)htmlをいったんローカルファイルに落として再度IEコンポーネントで処理する場合
などで
元ページのセキュリティレベルより低いゾーンで実行されてしまうという問題です。

KJ_browser.exe はIEコンポーネントを使用していますが
上記(1)(2)に該当するような処理は行っていません。

そのため本脆弱性はKJ_browser.exeでは該当しません。
念のため、IEの下にでるステータスバーのゾーン表示は常に確認ください。
元ページのゾーンと変わる事はないはずですが。
