프리(free)의 일한사전,한일사전 "KJ_dict"

간단한 일한 전자 사전을 만들 수 없을까하고 실험해 보았습니다.
정식 한국어사전을 만들 실력은 아직 없어서
일단은 제가 아는 것을 엮은 단어장 수준이 목표입니다.
일본어 학습용으로 어떻습니까?

         

특징

  • 일==>한,한==>일 의 쌍방향의 검색
  • 완전 일치, 전방 일치, 후방 일치, 부분 일치를 사용 할 수 있습니다
  • 히라가나, 한자, 한글 뿐만이 아니라 알파벳에서도 검색할 수 있습니다
  • 약 6만어의 데이터
  • 다채로운 인크리멘탈 검색(incremental search)
  • 사전 데이터와 표시 프로그램을 동고 있어, 곧바로 사용할 수 있습니다
  • 프리(free) : GPL
