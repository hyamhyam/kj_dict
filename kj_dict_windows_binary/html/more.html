特徴

  • 多彩なインクリメンタル検索。
    • ひらがな/漢字/ハングルでインクリメンタル検索。
    • アルファベット/ローマ字のままでもインクリメンタル検索。
    • ハングルの字母単位でのインクリメンタル検索も可。
    • 正規表現でインクリメンタル検索もできます。
  • 日韓辞書, 韓日辞書は区別しません。
  • フリー (GPL) です。


  • 以下は日本人のハングル学習者向け機能です
    1. 2ボル式のハングルIMEを同梱しています。
    2. ボタンを押す方式のハングルの簡易IME を同梱しています。


  • .NET framework 2.0以上 を前提とします。 (Windows Vista以降では最初から入っているので不要です)
