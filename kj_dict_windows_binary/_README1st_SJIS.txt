フリーの日韓辞書・韓日辞書 KJ_dict
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
使い方
・zipファイルを展開して 「KJ_dictform.exe」を実行。
  インストール不要です。
  またいらなくなった時はファイルを消すだけで構いません。
  （本ツールはレジストリやGACを触りません）


対象OS
・Windows 7/Vista/XP

本ツールの一次配布の場所
http://code.google.com/p/kjdict/downloads/list


注意事項
・初期版のWindows XPでは、本ツールの実行に .NET framework 2.0以降が必要です。
  以下よりdownloadして、先にインストールして下さい。
  http://www.microsoft.com/japan/msdn/netframework/downloads/
  ここの「.NET Framework 3.5 再頒布パッケージ」からdownloadして下さい。
  (これもフリーです)
  Windows Vista以降では.NET frameworkは既に入っていますから
  インストール不要です。

・プログラムとデータ、全てをGPLとします。
  またデータにはGPLであるFreeWnnの一部を改変して含んでいます。
  ライセンスに関してはsrcの下のCOPYING.txtを参照下さい。

・FreeWnnは、
  Xsi\kWnn\kdic\hword.u
  Xsi\kWnn\kdic\single.u
  の2つのファイルを利用しています。
  FreeWnnの1次配布先は以下になります。
  http://www.freewnn.org/

・アイコンはhttp://free-icons.jonasjohn.de/の
  フリーアイコンを使用させて頂いています。


データの出典
  FreeWnnのデータ以外に、
  いくつかの公開されているデータを作者の許可を得て使用しています。
  データの出典はKJ_dict.yml中のsrc:タグで識別できます。
    
  詳しくは
  html/data.html
  を参照ください。


ソースファイル
